# Система техпідтримки (Система пропусків) — Варіант №9

Це повноцінний Full-stack додаток для управління заявками в службу технічної підтримки. Проєкт реалізований за архітектурою **Controller-Service-Repository** з використанням TypeScript на бекенді та Vanilla JS на фронтенді.

## Основні можливості

- **Управління тікетами**: Створення, редагування, видалення та перегляд заявок.
- **Система користувачів**: Сутність `Users` з розподілом ролей (Admin, Support, Client).
- **Динамічні статуси**: Налаштування статусів (New, InProgress, Resolved тощо) з власними кольорами.
- **Система повідомлень**: Чат усередині кожного тікета для обговорення проблем.
- **Фільтрація та сортування**: Пошук за темою, фільтр за статусом та сортування за пріоритетом.
- **Сучасний інтерфейс**: Темна тема (Dark Mode) та кастомний дизайн полів вводу.

## Технологічний стек

- **Backend**: Node.js, Express, TypeScript.
- **Frontend**: HTML5, CSS3 (Modern Dark Theme), JavaScript (ES6+).
- **Інструменти**:
  - `uuid`: Для генерації унікальних ідентифікаторів.
  - `ESLint`: Для перевірки якості коду (налаштований для `.ts` та `.mts` файлів через `jiti`).
  - `ts-node`: Для запуску TypeScript коду без попередньої компіляції.

## Структура проєкту

```text
├── src/                    # Бекенд (TypeScript)
│   ├── controllers/        # Обробка HTTP-запитів
│   ├── services/           # Бізнес-логіка та валідація
│   ├── repositories/       # Робота з даними (In-memory storage)
│   ├── dtos/               # Опис структур даних (Data Transfer Objects)
│   ├── routes/             # Маршрутизація API
│   ├── middleware/         # Проміжне ПЗ (обробка помилок)
│   ├── utils/              # Допоміжні класи (ApiError)
│   └── index.ts            # Точка входу сервера
├── public/                 # Фронтенд (Static files)
│   ├── index.html          # Головна сторінка
│   ├── app.js              # Логіка взаємодії з API
│   └── styles.css          # Стилізація інтерфейсу
├── tsconfig.json           # Конфігурація TypeScript
└── eslint.config.ts        # Налаштування ESLint
